Sobre o Projeto

Plutão é um sistema operacional Unix-like projetado para ser leve, seguro e modular, com arquitetura simples e foco educacional e experimental.

Este documento contém informações essenciais para desenvolvedores contribuírem com o projeto.

Estrutura do Código

/plutao
    ├── src/            # Kernel e módulos
    ├── include/        # Headers e definições
    ├── boot/           # Bootloader / inicialização
    ├── fs/             # Sistema de arquivos
    ├── drivers/        # Drivers de hardware
    ├── user/           # Programas de espaço de usuário
    ├── docs/           # Documentação
    ├── scripts/        # Ferramentas de build
    ├── Makefile
    └── README.md

Instalação do Ambiente

Para compilar o Plutão, você precisará de:

  • GCC ou Clang
  • make
  • QEMU (para testes)
  • GNU Binutils

Clone o projeto:

git clone https://github.com/seuprojeto/plutao.git
cd plutao

Como Compilar

Após clonar o repositório:

Build padrão:

make

Executar no QEMU:

make run

Gerar imagem ISO:

make iso

Como Contribuir

1. Faça um fork do projeto
2. Crie uma branch:
   git checkout -b minha-feature
3. Faça seus commits:
   git commit -m "Descrição da alteração"
4. Envie para seu repositório:
   git push origin minha-feature
5. Abra um Pull Request
